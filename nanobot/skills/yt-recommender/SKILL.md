# Instruction Protocol

1. Pass the user's focus prompt to the `search_yt.py` background worker task script.
2. Read the structured list of titles and URLs generated by the tool.
3. Use your reasoning engine to select the best, high-value educational videos.
4. Output your response formatted cleanly with titles and active URL hyperlinks.
